# python-file-renamer
English

File Date Renamer is a Python tool for automatically renaming files based on their creation or modification date.

The application allows users to customize the naming format, including the position and formatting of the date within the filename. Subfolders can optionally be included or excluded from the renaming process.

After processing, the tool generates a detailed HTML report containing all renamed files. The report provides a clear overview of the performed changes and includes convenient features such as copying file paths directly from the report.

Deutsch

Der File Date Renamer ist ein Python-Tool zum automatischen Umbenennen von Dateien anhand ihres Erstellungs- oder Änderungsdatums.

Dabei können verschiedene Namensformate frei konfiguriert werden, beispielsweise die Position des Datums im Dateinamen oder dessen Formatierung. Zusätzlich können Unterordner bei der Verarbeitung wahlweise berücksichtigt oder ausgeschlossen werden.

Nach Abschluss der Umbenennung erstellt das Programm ein ausführliches HTML-Protokoll mit allen vorgenommenen Änderungen. Dieses bietet eine übersichtliche Darstellung der durchgeführten Umbenennungen und enthält praktische Funktionen wie das direkte Kopieren von Dateipfaden.
